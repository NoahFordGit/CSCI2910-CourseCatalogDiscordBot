# CSCI2910-CourseCatalogDiscordBot
Group project for ETSU CSCI 2910 Serverside Web Programming class.

# Description
A discord bot that uses an API rooted in a database containing courses and degrees from ETSU's CBAT department.  The bot enables users to find out information about any course within the CBAT catalogue.  The bot will list a description, credit hours, prerequisites, and what major/concentration the course is specific to.

# Tech Stack
Language: Python
Pydantic
Sqlalchemy
FastAPI
APIs: Database and API included

# Contributions
Bot can be adjusted to fit other degrees, courses, schools, etc.  With different databases and APIs.  
Could be added to to include the entirety of ETSU's programs.
